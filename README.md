## Power BI Dashboard

The dashboard was built using Power BI (online) and visualizes the cleaned e-commerce logistics dataset produced by the Python pipeline.

**Key features:**
- KPI cards for On-Time Delivery Rate and Average Delivery Days
- Monthly order volume trend
- Total delivery delays by customer region
